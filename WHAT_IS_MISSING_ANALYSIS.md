## 🔴 الناقص الحرج (Critical)

### 1. **واجهة المستخدم للميزات الجديدة**

**المشكلة:** الطبقات والمحركات موجودة لكن لا توجد واجهة مستخدم لاستخدامها!

**الناقص:**
- [ ] واجهة لـ Structured AI Response (الردود المنظمة)
- [ ] واجهة لـ Follow-Up Questions (الأسئلة المتابعة)
- [ ] واجهة لـ What-If Scenarios (السيناريوهات)
- [ ] واجهة لـ Predictions (التوقعات)
- [ ] واجهة لـ Recommendations (التوصيات)
- [ ] واجهة لـ Historical Context (السياق التاريخي)

**التأثير:** المستخدم لا يرى الميزات الجديدة رغم أنها موجودة!

**الجهد المطلوب:** 16-20 ساعة

---

### 2. **تكامل الواجهة مع الـ Backend**

**المشكلة:** الواجهات الموجودة لا تتصل بالـ tRPC procedures الجديدة!

**الناقص:**
- [ ] ربط NewFeaturesDashboard مع newFeaturesRouter
- [ ] ربط ResponseFeedback مع feedbackLoopSystem
- [ ] ربط TemporalComparison مع temporalAnalysisEngine
- [ ] ربط Daily Weather مع dailyEmotionalWeather
- [ ] ربط Universal Q&A مع universalQuestionAnswering

**التأثير:** الميزات موجودة لكن لا تعمل من الواجهة!

**الجهد المطلوب:** 12-16 ساعة

---

### 3. **نظام الإشعارات في الوقت الفعلي**

**المشكلة:** لا يوجد نظام إشعارات WebSocket!

**الناقص:**
- [ ] WebSocket server implementation
- [ ] Real-time alert system
- [ ] User notification preferences
- [ ] Alert triggers (GMI > 70, CFI < 30)
- [ ] Notification delivery (email, push, SMS)

**التأثير:** المستخدمون لا يعرفون عن التغييرات المهمة!

**الجهد المطلوب:** 20-24 ساعة
